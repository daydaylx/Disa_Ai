# Tab Navigation (Tabs)

**Pfad:** `src/components/ui/tabs.tsx`

Die Tab-Komponenten basieren auf Radix Tabs + Tokens:
- Hintergründe nutzen `bg-surface-subtle` / `bg-surface-card`
- Textfarben: `text-text-muted` → `text-text-primary` bei aktiven Tabs
- Fokus: `var(--color-border-focus)`

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from "../../src/components/ui/tabs";

export function TabNavDemo() {
  return (
    <Tabs defaultValue="chat">
      <TabsList aria-label="Bereiche">
        <TabsTrigger value="chat">Chat</TabsTrigger>
        <TabsTrigger value="roles">Rollen</TabsTrigger>
        <TabsTrigger value="models">Modelle</TabsTrigger>
      </TabsList>
      <TabsContent value="chat">Chat-Inhalt…</TabsContent>
      <TabsContent value="roles">Rollen-Inhalt…</TabsContent>
      <TabsContent value="models">Modelle-Inhalt…</TabsContent>
    </Tabs>
  );
}
```

## Tokens
| Bereich | Token |
| --- | --- |
| Hintergrund | `--color-surface-subtle`, `--color-surface-card` |
| Text | `--color-text-muted`, `--color-text-primary` |
| Border | `--color-border-subtle` |
| Fokus | `--color-border-focus` |

Die Komponenten lassen sich direkt in Storybook importieren und benötigen keinerlei zusätzlichen Zustand außer den Radix-Props.
