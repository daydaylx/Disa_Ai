# Legacy Hooks

Diese Hooks werden aktuell nicht verwendet, wurden aber aufbewahrt.

## Status: DEPRECATED (November 2025)

| Hook               | Beschreibung                | Grund für Deprecation             |
| ------------------ | --------------------------- | --------------------------------- |
| `useChatMemory.ts` | Memory-Integration für Chat | Nicht integriert                  |
| `useEdgeSwipe.ts`  | Edge-Swipe-Gesten           | Ersetzt durch `useBookNavigation` |
| `useMediaQuery.ts` | Media Query Hook            | Tailwind-Breakpoints bevorzugt    |
| `useRoles.ts`      | Rollen-Management           | Ersetzt durch `RolesContext`      |

## Löschung

Diese Dateien können nach 3 Monaten (Februar 2026) gelöscht werden.
